Employee management system:
 
introduction:
The employee management system is a python based system that is used to store and manipulate the employee data.
It is a command line based management system thorugh wich we can add, delete, search and update the employee records


#features:
In this system we can add the employee records such as employee id,name,age,gender,date of joining, date of birth, department name, designation,etc.
we can also update the employee reords such as particular employee salary, update salary of employee department wise, update the name of employee, etc
we can also serach the employee by name or employee id or department of employee.
In this system we can also delete the Employee record and display the details of employee with maximum salary
